# SkillBridge — منصة ربط الطلاب بسوق العمل

منصة تعليمية بـ ASP.NET Core 9 (MVC) بثلاثة أدوار: **مدير (Admin)**، **طالب (Student)**، **شركة (Company)**.

---

## كيف تشغّلي المشروع خطوة خطوة (VS Code)

### 1) المتطلبات
لازم يكون عندك **.NET 9 SDK** مثبّت. تأكدي بكتابة:

```
dotnet --version
```

لو طلع رقم يبدأ بـ 9 → تمام. لو لأ، نزّليه من:
https://dotnet.microsoft.com/download/dotnet/9.0

### 2) افتحي المشروع
- افتحي مجلد `SkillBridge` كامل في VS Code.
- افتحي Terminal داخل VS Code (Terminal → New Terminal).

### 3) نزّلي الحزم
```
dotnet restore
```

### 4) شغّلي المشروع
```
dotnet run
```

أول مرة رح ينشئ قاعدة البيانات تلقائياً (`skillbridge.db`) ويعبّيها ببيانات تجريبية + حساب المدير.

### 5) افتحي المتصفح
روحي على:
```
http://localhost:5000
```

## حسابات الدخول

### حساب المدير (جاهز مسبقاً)
```
admin@skillbridge.com
Admin@123
```

### حساب طالب أو شركة
سجّلي حساب جديد من صفحة "إنشاء حساب" واختاري النوع (طالب/شركة).

---

## وش بيعمل كل دور؟

**المدير**: يضيف التخصصات، المراحل، الروابط/الفيديوهات، الأسئلة، ويتابع كل الطلاب والشركات.

**الطالب**: يختار تخصص → يشوف المراحل والمصادر → يحل الاختبار → ياخد نقاط ويطلعله سكور.

**الشركة**: تتصفّح التخصصات والطلاب حسب سكورهم وتتواصل معهم بالإيميل.

---

## ملاحظات

- قاعدة البيانات **SQLite** (ملف `skillbridge.db`) — ما بتحتاجي أي إعداد إضافي.
- لو بدك تصفّري كل البيانات: احذفي ملف `skillbridge.db` وشغّلي `dotnet run` من جديد.
- الألوان والتصميم في `wwwroot/css/site.css`.
